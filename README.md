# FIRST-Discovery

The Folders in this repository are intended to get you started thinking about the possiblities that Misty offers, specifically how you can being programming a robot utilizing JavaScript. The best way to start is to download this repository locally on your machine and begin with Activity 1. 

We'll begin with some basics like facial identification and proceed with giving you the ability to modify Misty's behaviour with movement, sounds, and being able to recognize individuals!
